# **App Name**: Firebase Studio Handbook

## Core Features:

- Feature List Display: Display a comprehensive checklist of Firebase Studio features in a left-side panel.
- JSON Data Loading: Load feature details, including text and image paths, from a JSON file.
- Dynamic Image Replacement: Replace image path placeholders (e.g., [img/path.jpg]) in the loaded JSON data with actual image elements.
- Feature Information Display: Upon clicking a feature in the checklist, display detailed information about its implementation and usage within the application.

## Style Guidelines:

- Primary color: A soft blue (#A0D2EB) to reflect the calming nature of clear documentation.
- Background color: A very light gray (#F0F0F0) to provide a clean and unobtrusive backdrop.
- Accent color: A vibrant orange (#FFB347) to highlight key interactive elements and call-to-action areas.
- Headline Font: 'Poppins', a geometric sans-serif for headlines.
- Body Font: 'PT Sans', a humanist sans-serif for body text.
- Use minimalist, line-style icons to represent each Firebase feature in the checklist.
- Employ a split-screen layout with the feature checklist on the left and detailed information on the right.
